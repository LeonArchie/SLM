Файл 50x.css содержит стили для страницы, которая отображается при возникновении ошибок сервера 
(например, ошибки 500, 502, 503 и т.д.). Этот файл отвечает за визуальное оформление страницы, 
чтобы пользователь мог понять, что произошла ошибка, и получить информацию о дальнейших действиях. 

Основные стили:
    Общие стили для страницы (body):
        font-family: Устанавливает шрифт Arial (или sans-serif, если Arial недоступен) для всей страницы.
        background-color: Фон страницы имеет светло-серый цвет (#f4f4f9).
        color: Основной цвет текста — темно-серый (#333).
        margin и padding: Убираются отступы по умолчанию, чтобы страница выглядела аккуратно.
        display: flex: Используется Flexbox для центрирования содержимого страницы по вертикали и горизонтали.
        justify-content: center и align-items: center: Центрирует содержимое по центру страницы.
        height: 100vh: Задает высоту страницы равной высоте окна браузера (100% viewport height).
        text-align: center: Выравнивает текст по центру.

    Контейнер для содержимого (.container):
        max-width: 600px: Ограничивает ширину контейнера до 600 пикселей, чтобы содержимое не растягивалось на всю ширину экрана.
        padding: 20px: Добавляет внутренние отступы вокруг содержимого контейнера.
        background-color: #fff: Фон контейнера — белый.
        border-radius: 10px: Закругляет углы контейнера.
        box-shadow: Добавляет тень вокруг контейнера для эффекта "поднятости".
    
    Заголовок (h1):
        font-size: 3rem: Устанавливает крупный размер шрифта для заголовка.
        margin: 0: Убирает внешние отступы у заголовка.
        color: #e74c3c: Заголовок окрашен в красный цвет, чтобы привлечь внимание пользователя к ошибке.

    Текст (p):
        font-size: 1.2rem: Устанавливает размер шрифта для текста. 
        margin: 20px 0: Добавляет отступы сверху и снизу для текста.

    Ссылка (a):
        color: #3498db: Ссылки окрашены в синий цвет.
        text-decoration: none: Убирает подчеркивание у ссылок.
        font-weight: bold: Делает текст ссылок полужирным.

    Эффект при наведении (a:hover):
        text-decoration: underline: При наведении на ссылку появляется подчеркивание, чтобы пользователь понимал, что это кликабельный элемент.
