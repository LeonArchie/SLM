Файл all.css содержит основные стили для веб-приложения. 
Он определяет общее оформление страницы, включая шапку (header), 
основное содержимое (main) и подвал (footer). Этот файл обеспечивает 
единообразие стилей для всех страниц приложения. 

Основные стили:
    Общие стили для страницы (body):
        display: flex: Используется Flexbox для центрирования содержимого страницы.
        justify-content: center: Центрирует содержимое по горизонтали.
        align-items: center: Центрирует содержимое по вертикали.
        height: calc(100vh - 3em - 2em): Высота страницы рассчитывается как высота окна браузера минус высота шапки (3em) и подвала (2em).
        margin: 0: Убирает внешние отступы по умолчанию.
        background-color: #fdf5e6: Фон страницы имеет молочный цвет.
        font-size: 16px: Устанавливает базовый размер шрифта.
        font-family: Arial, sans-serif: Устанавливает шрифт Arial (или sans-serif, если Arial недоступен).
    
    Шапка (.header):
        display: flex: Использует Flexbox для размещения элементов шапки в одной строке.
        align-items: center: Выравнивает элементы шапки по вертикали по центру.
        padding: 10px 20px: Добавляет внутренние отступы.
        position: fixed: Фиксирует шапку в верхней части страницы.
        top: 0px, left: 0px, right: 0px: Растягивает шапку на всю ширину страницы.
        height: 2em: Устанавливает высоту шапки.
        z-index: 1000: Убеждается, что шапка находится поверх других элементов.
        background-color: #6b7a8f: Фон шапки имеет серо-голубой цвет.
        border: 2px solid #333: Добавляет рамку вокруг шапки.
    
    Контейнер для логотипа (.logo-container):
        text-align: center: Центрирует содержимое контейнера.
        padding: 10px: Добавляет внутренние отступы.
        margin-right: 20px: Добавляет отступ справа для разделения логотипа и текста.

    Логотип (.logo-image):
        width: auto: Ширина логотипа автоматически подстраивается под содержимое.
        height: 2.5em: Устанавливает высоту логотипа.

    Название приложения (.name):
        text-decoration: none: Убирает подчеркивание у текста.
        color: inherit: Наследует цвет текста от родительского элемента.
        position: relative: Устанавливает относительное позиционирование.
        left: 10px: Сдвигает текст вправо на 10 пикселей.
        font-size: 1.5em: Устанавливает размер шрифта.
        margin: 0: Убирает внешние отступы.

    Основное содержимое (main):
        padding-top: 6em: Добавляет отступ сверху, чтобы содержимое не перекрывалось шапкой.
        padding-bottom: 6em: Добавляет отступ снизу, чтобы содержимое не перекрывалось подвалом.

    Подвал (.footer):
        display: flex: Использует Flexbox для размещения элементов подвала.
        flex-direction: column: Элементы подвала располагаются в колонку.
        justify-content: center: Центрирует элементы по вертикали.
        align-items: center: Центрирует элементы по горизонтали.
        padding: 10px 20px: Добавляет внутренние отступы.
        position: fixed: Фиксирует подвал в нижней части страницы.
        bottom: 0, left: 0, right: 0: Растягивает подвал на всю ширину страницы.
        height: auto: Высота подвала автоматически подстраивается под содержимое.
        width: 100%: Ширина подвала равна ширине страницы.
        background-color: #f8f9fa: Фон подвала имеет светло-серый цвет.
        border-top: 1px solid #ddd: Добавляет границу сверху.

    Версия приложения (.version):
        font-size: 0.7em: Устанавливает небольшой размер шрифта.
        color: #666: Текст имеет серый цвет.
        text-align: center: Выравнивает текст по центру.
        margin: 1px 0: Добавляет небольшие отступы сверху и снизу.
