CSS-файлы  используются для стилизации различных компонентов веб-приложения. 
Они обеспечивают единообразие дизайна, удобство взаимодействия с пользователем и 
визуальную привлекательность. 

1. all.css
    Назначение: Основной файл стилей, который задает общие правила для всего приложения.
    Что включает:
        Стили для шапки (header), основного содержимого (main) и подвала (footer).
        Центрирование содержимого страницы.
        Фиксированное положение шапки и подвала.
        Фон страницы (молочный цвет) и шрифты (Arial, sans-serif).
        Отступы для основного содержимого, чтобы оно не перекрывалось шапкой и подвалом.
    Роль: Обеспечивает единый стиль для всех страниц приложения.

2. login.css
    Назначение: Стили для страницы авторизации (входа в систему).
    Что включает:
        Оформление формы входа: поля для логина и пароля, кнопка отправки.
        Эффекты при наведении на кнопку.
        Сообщения об ошибках (например, "Неверный логин или пароль").
        Центрирование формы на странице.
    Роль: Делает страницу входа интуитивно понятной и визуально привлекательной.

3. navbar.css
    Назначение: Стили для навигационной панели (navbar).
    Что включает:
        Оформление ссылок навигации.
        Выпадающие меню для дополнительных разделов (например, "Учетные записи").
        Эффекты при наведении на ссылки.
        Фиксированное положение панели в верхней части страницы.
        Тень и закругленные углы для визуального выделения.
    Роль: Обеспечивает удобную навигацию по приложению.

4. register.css
    Назначение: Стили для страницы регистрации пользователя.
    Что включает:
        Оформление формы регистрации: поля для ввода email, логина, пароля, выбора роли.
        Кнопку генерации пароля.
        Индикаторы валидации (зеленый/красный кружок для отображения состояния полей).
        Сообщения об ошибках и уведомления об успешной регистрации.
        Эффекты при наведении на кнопки.
    Роль: Упрощает процесс регистрации и делает его удобным для пользователя.

5. 50x.css
    Назначение: Стили для страницы, отображаемой при ошибках сервера (например, 500 Internal Server Error).
    Что включает:
        Центрирование содержимого страницы.
        Красный заголовок для привлечения внимания.
        Ссылку на главную страницу.
        Тень и закругленные углы для контейнера с сообщением.
    Роль: Информирует пользователя об ошибке и предлагает вернуться на главную страницу.

6. 403.css
    Назначение: Стили для страницы, отображаемой при ошибке доступа (403 Forbidden).
    Что включает:
        Центрирование содержимого страницы.
        Красный заголовок для привлечения внимания.
        Ссылку на главную страницу.
        Тень и закругленные углы для контейнера с сообщением.
    Роль: Информирует пользователя о запрете доступа и предлагает вернуться на главную страницу.

7. 404.css
    Назначение: Стили для страницы, отображаемой при ошибке "Страница не найдена" (404 Not Found).
    Что включает:
        Центрирование содержимого страницы.
        Красный заголовок для привлечения внимания.
        Ссылку на главную страницу.
        Тень и закругленные углы для контейнера с сообщением.
    Роль: Информирует пользователя о том, что запрашиваемая страница не существует, и предлагает вернуться на главную страницу.
