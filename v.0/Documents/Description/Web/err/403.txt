Файл 403.html представляет собой HTML-страницу, которая отображается при 
попытке пользователя получить доступ к ресурсу, для которого у него нет прав (
ошибка 403 Forbidden). Эта страница информирует пользователя о запрете доступа и 
предлагает возможные действия для решения проблемы.

Структура файла:
    Заголовок (<head>):
        <title>SLM</title>: Заголовок страницы, отображаемый вкладке браузера.
        <meta charset="utf-8">: Указывает кодировку страницы (UTF-8).
        <meta name="description" content="Управление жизненным циклом серверов и приложений">: Описание страницы для поисковых систем.
        <meta name="robots" content="noindex, nofollow">: Запрещает поисковым системам индексировать эту страницу.
        <link rel="stylesheet" href="/css/404.css">: Подключает файл стилей 404.css для оформления страницы.
        <link rel="icon" sizes="16x16 32x32 48x48" type="image/png" href="/img/icon.png">: Подключает фавиконку (иконку сайта).
    Тело страницы (<body>):
        <div class="container">: Контейнер для содержимого страницы.
        <h1>403</h1>: Заголовок страницы, который отображает код ошибки (403).
        <p>Упс! Доступ закрыт заботливым администратором.</p>: Сообщение о запрете доступа.
        <p>Попробуйте вернуться на <a href="/">главную страницу</a>.</p>: Предложение пользователю вернуться на главную страницу. Ссылка ведет на корневой URL (/).
