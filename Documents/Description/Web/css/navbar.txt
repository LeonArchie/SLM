Файл navbar.css содержит стили для навигационной панели (navbar) веб-приложения.
Навигационная панель обычно располагается в верхней части страницы и предоставляет 
пользователю доступ к различным разделам приложения. Этот файл отвечает за оформление 
навигационной панели, включая ссылки, выпадающие меню и эффекты при наведении.

Основные стили:
    Навигационная панель (.navbar):
        background-color: #7CB68C: Фон панели — светло-зеленый.
        padding: 3px 15px: Добавляет внутренние отступы.
        position: fixed: Фиксирует панель в верхней части страницы.
        top: 3.48em: Располагает панель ниже шапки (header).
        left: 0, right: 0: Растягивает панель на всю ширину страницы.
        z-index: 999: Убеждается, что панель находится поверх других элементов.
        box-shadow: 0 2px 5px rgba(0, 0, 0, 0.2): Добавляет тень для эффекта "поднятости".

    Список ссылок (.nav-links):
        list-style: none: Убирает маркеры списка.
        display: flex: Использует Flexbox для размещения элементов в строке.
        margin: 0, padding: 0: Убирает внешние и внутренние отступы.
        align-items: center: Выравнивает элементы по вертикали по центру.
    
    Элементы списка (.nav-links li):
        margin-left: 10px: Добавляет отступ между пунктами меню.
        position: relative: Устанавливает относительное позиционирование для выпадающего меню.
    
    Элемент, выравненный по правому краю (.nav-links li.push-right):
        margin-left: auto: Автоматически добавляет отступ слева, чтобы элемент был выровнен по правому краю.

    Ссылки (.nav-links a):
        color: #000000: Цвет текста — черный.
        text-decoration: none: Убирает подчеркивание у ссылок.
        font-size: 1em: Устанавливает размер шрифта.
        padding: 8px 10px: Добавляет внутренние отступы.
        transition: background-color 0.3s ease: Добавляет плавное изменение цвета фона при наведении.
        border-radius: 5px: Закругляет углы ссылок.
        display: flex: Использует Flexbox для выравнивания иконок и текста.
        align-items: center: Выравнивает иконки и текст по вертикали.
    
    Эффект при наведении на ссылки (.nav-links a:hover):
        background-color: #f5f5f55e: Изменяет цвет фона ссылки при наведении.
    
    Активная ссылка (.nav-links a.active):
        background-color: #f5f5f55e: Устанавливает цвет фона для активной ссылки.
        font-weight: bold: Делает текст полужирным.

    Выпадающее меню (.dropdown-menu):
        display: none: Скрывает выпадающее меню по умолчанию.
        position: absolute: Устанавливает абсолютное позиционирование.
        top: 100%: Располагает меню ниже родительского элемента.
        left: 0: Выравнивает меню по левому краю.
        background-color: #7CB68C: Фон меню — светло-зеленый (как у навигационной панели).
        list-style: none: Убирает маркеры списка.
        margin: 0, padding: 0: Убирает внешние и внутренние отступы.
        min-width: 200px: Устанавливает минимальную ширину меню.
        border-radius: 5px: Закругляет углы меню.
        box-shadow: 0 4px 10px rgba(0, 0, 0, 0.2): Добавляет тень.
        z-index: 1000: Убеждается, что меню находится поверх других элементов.
    
    Элементы выпадающего меню (.dropdown-menu li):
        margin: 0: Убирает отступы между пунктами.
    
    Ссылки в выпадающем меню (.dropdown-menu a):
        padding: 10px 20px: Добавляет внутренние отступы.
        display: block: Ссылки занимают всю ширину.

    Показ выпадающего меню при наведении (.dropdown:hover .dropdown-menu):
        display: block: Показывает выпадающее меню при наведении на родительский элемент.
