
Файл 50x.html представляет собой HTML-страницу, которая отображается при возникновении ошибок сервера 
(например, 500 Internal Server Error). Эта страница информирует пользователя о проблеме и предлагает 
возможные действия для её решения.

Структура файла:
    Заголовок (<head>):
        <title>SLM</title>: Заголовок страницы, отображаемый вкладке браузера.
        <meta charset="utf-8">: Указывает кодировку страницы (UTF-8).
        <meta name="description" content="Управление жизненным циклом серверов и приложений">: Описание страницы для поисковых систем.
        <meta name="robots" content="noindex, nofollow">: Запрещает поисковым системам индексировать эту страницу.
        <link rel="stylesheet" href="/css/50x.css">: Подключает файл стилей 50x.css для оформления страницы.
        <link rel="icon" sizes="16x16 32x32 48x48" type="image/png" href="/img/icon.png">: Подключает фавиконку (иконку сайта).
    Тело страницы (<body>):
        <div class="container">: Контейнер для содержимого страницы.
        <h1>Ошибка сервера</h1>: Заголовок страницы, выделенный красным цветом (стили задаются в 50x.css).
        <p>Упс! Что-то пошло не так на стороне сервера.</p>: Сообщение об ошибке.
        <p>Пожалуйста, попробуйте обновить страницу позже или вернитесь на <a href="/">главную страницу</a>.</p>: 
            Предложение пользователю обновить страницу или вернуться на главную страницу. Ссылка ведет на корневой URL (/).
