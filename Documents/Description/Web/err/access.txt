Файл access.html представляет собой HTML-страницу, которая отображается, 
когда пользователь пытается получить доступ к странице к которой у него нет
доступа по ролевой модели. Эта страница информирует пользователя о том, 
что доступа нет, и предлагает возможные действия для решения проблемы.

Структура файла:
    Заголовок (<head>):
        <title>SLM</title>: Заголовок страницы, отображаемый вкладке браузера.
        <meta charset="utf-8">: Указывает кодировку страницы (UTF-8).
        <meta name="description" content="Управление жизненным циклом серверов и приложений">: Описание страницы для поисковых систем.
        <meta name="robots" content="noindex, nofollow">: Запрещает поисковым системам индексировать эту страницу.
        <link rel="stylesheet" href="/css/access.css">: Подключает файл стилей access.css для оформления страницы.
        <link rel="icon" sizes="16x16 32x32 48x48" type="image/png" href="/img/icon.png">: Подключает фавиконку (иконку сайта).
    Тело страницы (<body>):
        <div class="container">: Контейнер для содержимого страницы.
        <h1>404</h1>: Заголовок страницы, который отображает ошибку доступа
        <p>Доступ к данной странице закрыт.</p>: Сообщение о том, что доступ закрыт.
        <p>Запросите доступ у администратора системы. Попробуйте вернуться на <a href="/">главную страницу</a>.</p>
            Предложение пользователю вернуться на главную страницу. Ссылка ведет на корневой URL (/).
